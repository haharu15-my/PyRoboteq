## main
卒業研究で使用したメインプログラム。

本研究では、ロボット走行中に発生する**スタック（走行不能状態）**を検知し、  
自律的に脱出動作を行う制御を実装している。

しかし今はキーボード操作によるロボット制御とスタック検知・脱出制御にとどまっている

スタックは以下の状態を基準に判定する。

- モータ回転数（RPM）がほぼ0になる  
- モータ電流が高い状態で張り付き、変動が小さい

これらの条件が一定時間継続した場合にスタックと判定し、  
**通常走行用の駆動指令から脱出用の駆動指令へ切り替えることで、  
一時的に大きな駆動力を確保してスタックからの脱出を試みる。**

脱出動作後に回転が回復すれば通常走行へ復帰し、  
回復しない場合は安全のため停止する。
